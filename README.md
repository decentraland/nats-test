# Nats test

Run `go run main.go`

```
  -debug
    	Print debug info
  -islands int
    	Total islands to distribute peers between (default 10)
  -peers int
    	Total peers to simulate (default 10000)
  -start int
    	Start from this peer number
  -url string
    	Nats cluster URL (default "nats://127.0.0.1:4222")
```
